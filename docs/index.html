VR Avatar Viewer とは

アバターを「読み込んで眺める」だけでなく、ポーズ、表情、ライト、カメラ、背景、シーケンス演出まで扱えることを目標に開発しています。

  • VRM / VAB対応
    複数形式のアバターを読み込み、管理します。
  • デスクトップ / VR
    マウス操作とVRコントローラー操作の両方に対応します。
  • ポーズ・表情編集
    指、ボーン、BlendShape、パーツ表示を調整します。
  • 撮影環境
    ライト、物理カメラ、背景、ポストエフェクトを扱います。
  • 独自フォーマット
    VAB、VAC、VBG、VSP、VASを用途ごとに設計しています。
  • 演出と再生
    アニメーションやファッションショー形式の再生を目指しています。
